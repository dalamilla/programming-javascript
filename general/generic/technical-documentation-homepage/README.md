# Technical Documentation Home Page

Technical documentation homepage.

## Instructions

To start this app:

- Installing Dependencies:

```
npm i
```

- Compiles and hot-reload for development:

```
npm run dev
```

- Compiles and minifies for production:

```
npm run build
```

- Run tests:

```
npm run test:e2e
```
