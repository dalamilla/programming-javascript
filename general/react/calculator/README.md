# Calculator

Calculator app.

## Instructions

To start this app:

- Installing Dependencies:

```
npm ci
```

- Compiles and hot-reloads for development:

```
npm run dev
```

- Compiles and minifies for production:

```
npm run build
```

- Run unit tests:

```
npm run test:unit
```

- Run lint:

```
npm run lint
```

- Check format files:

```
npm run format
```
